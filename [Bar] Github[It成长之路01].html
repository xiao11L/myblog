Github 成 长 之 路

一、Github注册安装

Github官网：https://github.com 可直接进入进行注册登录，不需要科学上网。
Github百科介绍：https://baike.baidu.com/item/github/10145341?fr=aladdin


登录后点击New--->输入Repository name（就是你的创建项目名称）--->输入Description (optional)（就是介绍项目）--->勾选 Initialize this repository with a README --->点击Create

二、Git下载

各系统下载地址：https://git-scm.com/downloads 速度可能较慢可以科学上网后加速下载，下载后默认配置安装即可

三、使用Git配置发布静态网页

视频教程：B站Git教学10分钟，跟着敲就行，最后出错可以看看评论区增加自己思考能力。

命令:
· cd （返回根目录）
· ls (查看目录文件)
· mkdir *** (***为所需要创建的文件名)
· cd *** (进入创建的文件夹)
· git config --global user.name "***" （***为你的Github用户名）
· git config --global user.email "***" （***为你的Github注册所用的邮箱）
· git init (设定此文件夹根处为部署位置，同时把需要发布的html文件放在此文件夹中，一般在C盘-用户-计算机名-文件夹名)
· git status (查看文件夹中文件)
· git add . (添加创建文件夹中的所有文件，若把.改为*效果为添加文件夹中所有修改过的文件)
· git status (查看文件夹中添加后文件)
· git commit -m "***" (***为此次添加提交的备注，提供一个commit)
· git status (查看文件夹中commit后的文件)
· git remote ()
· git remote add origin https://github.com/xiao11L/login.git (origin后的地址为新建的项目Clone and download按钮中的地址，不是usr shh的地址)
· git remote -v (显示所有远程仓库)
· git pull --rebase origin master(将文件推到本地)
· git push origin master （将本地文件推到github中）
· 完成~~可在Github项目中找到提交的html文件，在Setting中下方找到Github Pages中的None选择第一个后出现一个链接，转到后可看到html文件的预览

四、Git更新已经提交的项目

命令：
· git cd （返回根目录）
· git ls （查看目录文件夹）
· git add *（添加变动的文件）
· git commit -m "**" （提交commit 附加备注）
· git remote add origin https://github.com/xiao11L/login.git （添加至你的仓库）
· git pull --rebase origin master （将文件提交至本地）
· git push origin master （将文件提交至github中）
· 更新成功
